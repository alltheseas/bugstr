### What is it?
Automated, private crash reports via nostr transport layer.

### Where does it come from?
This approach was invented by [Vitor](https://github.com/vitorpamplona/) & team [Amethyst](https://github.com/vitorpamplona/amethyst) 

### Which OS?
Android.

### What's a nostr?
For a protocol overview see NIP-01.
For DMs approach and cryptography see [NIP-17](https://github.com/nostr-protocol/nips/blob/master/17.md) and [NIP-44](https://github.com/nostr-protocol/nips/blob/master/44.md).
